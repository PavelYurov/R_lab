в папке chans algorithms-6 находятся все имплементации алгоритма Чена (для 2d и 3d случая).
 в папке можете найти исходные файлы и все необходиые зависимости.

в папке visualization находятся файлы, позволяющие удостовериться в правильности работы кода.
файл  chan3d.ipynb - визуализация на питоне. также для проверки правильности работы предоставлен код, для построения оболочки встроенной функцией python
файлы chan3d-vis-preparata.exe, chan3d-vis-quickhull.exe,
chan3d-vis-preparata-input.exe, chan3d-vis-quickhull-input.exe - уже скомпилированные имплементации.

приписка -input значит, что ввод происходит вручную, через консоль. без приписки - ввод случайных точек.

файлы in.txt out.txt - файлы, в которые записываются входные и выходные точки. важно, чтоб все файлы
(chan3d-vis.exe, chan3d.ipynb, in.txt out.txt) были в одной директории.

файл chan3d-vis.cpp - код с выводом точек в файл, а не только в консоль.
